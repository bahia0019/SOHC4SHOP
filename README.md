# SOHC4SHOP

Static HTML/CSS site for **SOHC4SHOP**.

This repo is intentionally simple: no build tools, no frameworks, no backend. Just files that render in a browser, like the internet used to.

---

## Tech Stack

- **HTML**
- **CSS** (`stylesheet.css`)
- **JavaScript**
- **jQuery** (FancyBox)

---

## Hosting & Deployment

### Cloudflare Pages

This site is hosted on **Cloudflare Pages**.

- Deploys are triggered via **GitHub integration** (push to the main branch).
- The Pages project is configured in the Cloudflare dashboard.

### Domain & DNS

- **Domain registration:** Cloudflare
- **DNS:** Cloudflare
- **Custom domain:** attached to the Cloudflare Pages project

---

## Local Development

Because this is static, “local dev” is basically “open the file.”
